一小时搭起一个个人博客

如果你的电脑系统是Windows，那么请做好每一步都有可能报错的心理准备，当然也是有几率顺利完成的；如果你是Linux系统或Mac OS，那么跟着这个教程大概一个小时之后你就可拥有自己的个人博客了～

零、换系统

如果你是一名Windows系统使用者，想要体验Windows之外的其他系统，比如基于Linux的Ubuntu系统或者Deepin，请参考以下两篇教程：

一、准备工作

  1. 注册GitHub账号
    GitHub，点击这个链接或者在搜索引擎输入GitHub，进入GitHub主页，输入你的用户名、邮箱以及密码，注册好你的GitHub之后就放在那里我们等会再用，或者可以去搜索引擎学习一下GitHub的用法。
  2. 安装Git Bash
    Git Bash是Windows下的命令行工具，Windows自带的命令行叫cmd，如果不想被气到吐血请安装Git Bash谢谢。
    Git Bash-官方网站，当然了，由于我们大家都知道的原因，国内访问国外的网站大多数时候会出现一定的问题，在你不会科学上网的情况下，百度网盘：https://pan.baidu.com/s/1nu99KWp，密码：jfdf
